## English

An Obsidian plugin that organizes sidebar icons into grouped hover menus - declutter your sidebar by grouping plugin icons with hover-to-expand functionality.

### Features

- **Custom Groups** - Create custom groups and manually select which sidebar icons to merge
- **Custom Icons** - Set custom SVG icons for each group
- **Blur Effect** - Beautiful blur background effect for popup menus
- **Drag & Drop** - Reorder groups by dragging
- **Multi-language** - Supports 8 languages: English, 中文, 日本語, 한국어, Deutsch, Русский, Español, Français

### Usage

1. Open Settings → Sidebar Organizer
2. Click "Create Group" to create a new group
3. Select the sidebar icons you want to merge
4. Set a name and optional custom icon for the group
5. Hover over the merged icon to see the function menu

### Installation

#### From Obsidian Community Plugins (Recommended)
1. Open Settings → Community plugins
2. Disable "Restricted mode"
3. Click "Browse" and search for "Sidebar Organizer"
4. Click "Install" and then "Enable"

#### Manual Installation
1. Download `main.js`, `styles.css`, and `manifest.json` from [Releases](https://github.com/zhywjml/obsidian-sidebar-organizer/releases)
2. Create a folder named `sidebar-organizer` in your vault's `.obsidian/plugins/` directory
3. Copy the downloaded files into that folder
4. Restart Obsidian and enable the plugin in Settings → Community plugins

## 中文

一个 Obsidian 插件，将侧边栏图标整理成分组的悬停菜单 - 通过分组插件图标并使用悬停展开功能来清理您的侧边栏。

### 功能特点

- **自定义分组** - 创建自定义分组，手动选择要合并的侧边栏图标
- **自定义图标** - 为每个分组设置自定义 SVG 图标
- **毛玻璃效果** - 弹出菜单的美观模糊背景效果
- **拖拽排序** - 通过拖拽重新排列分组
- **多语言** - 支持 8 种语言：English, 中文, 日本語, 한국어, Deutsch, Русский, Español, Français

### 使用方法

1. 打开设置 → Sidebar Organizer
2. 点击"创建分组"创建新分组
3. 选择要合并的侧边栏图标
4. 设置分组名称和可选的自定义图标
5. 悬停在合并后的图标上即可看到功能菜单

### 安装

#### 从 Obsidian 社区插件安装（推荐）
1. 打开设置 → 第三方插件
2. 关闭"安全模式"
3. 点击"浏览"并搜索"Sidebar Organizer"
4. 点击"安装"，然后"启用"

#### 手动安装
1. 从 [Releases](https://github.com/zhywjml/obsidian-sidebar-organizer/releases) 下载 `main.js`、`styles.css` 和 `manifest.json`
2. 在您的 vault 的 `.obsidian/plugins/` 目录下创建名为 `sidebar-organizer` 的文件夹
3. 将下载的文件复制到该文件夹
4. 重启 Obsidian 并在设置 → 第三方插件中启用该插件

## 日本語

サイドバーアイコンをグループ化されたホバーメニューに整理する Obsidian プラグイン - プラグインアイコンをグループ化し、ホバーで展開する機能でサイドバーをすっきりさせます。

### 機能

- **カスタムグループ** - カスタムグループを作成し、統合するサイドバーアイコンを手動で選択
- **カスタムアイコン** - 各グループにカスタム SVG アイコンを設定
- **ブラー効果** - ポップアップメニューの美しい背景ブラー効果
- **ドラッグ＆ドロップ** - ドラッグでグループの順序を変更
- **多言語対応** - 8言語をサポート: English, 中文, 日本語, 한국어, Deutsch, Русский, Español, Français

### 使い方

1. 設定 → Sidebar Organizer を開く
2. 「グループ作成」をクリックして新しいグループを作成
3. 統合するサイドバーアイコンを選択
4. グループ名とオプションのカスタムアイコンを設定
5. 統合されたアイコンにホバーすると機能メニューが表示されます

## 한국어

사이드바 아이콘을 그룹화된 호버 메뉴로 정리하는 Obsidian 플러그인 - 플러그인 아이콘을 그룹화하고 호버로 펼치기 기능으로 사이드바를 정리하세요.

### 기능

- **사용자 정의 그룹** - 사용자 정의 그룹을 생성하고 통합할 사이드바 아이콘 직접 선택
- **사용자 정의 아이콘** - 각 그룹에 사용자 정의 SVG 아이콘 설정
- **블러 효과** - 팝업 메뉴의 아름다운 배경 블러 효과
- **드래그 앤 드롭** - 드래그로 그룹 순서 변경
- **다국어 지원** - 8개 언어 지원: English, 中文, 日本語, 한국어, Deutsch, Русский, Español, Français

### 사용 방법

1. 설정 → Sidebar Organizer 열기
2. "그룹 생성"을 클릭하여 새 그룹 생성
3. 통합할 사이드바 아이콘 선택
4. 그룹 이름과 선택적 사용자 정의 아이콘 설정
5. 통합된 아이콘에 마우스를 올리면 기능 메뉴가 표시됩니다
